# AI RAG Agent Frontend

Vue 3 frontend for the AI RAG Agent FastAPI service.

## Local Development

```powershell
npm install
npm run dev
```

The default API address is:

```text
/api
```

Override it with:

```text
VITE_API_BASE_URL=/api
```

## Build

```powershell
npm run build
```
